# Data Folder

Upload Bay Wheels trip data ZIP files here to automatically rebuild the dashboard.

The GitHub Actions workflow will detect new files and run the preprocessing pipeline.

Download data from: https://www.lyft.com/bikes/bay-wheels/system-data

## Current Data

The dashboard currently includes data through **January 2026** (`202601-baywheels-tripdata`). This is the most recent file available as of February 2026.
